# 🏴‍☠️ Super Pirate World

Super Pirate World is a 2D adventure/platformer game made with **Unity**.  
Play as a daring pirate exploring various levels, overcoming obstacles, and defeating enemies to find treasure.

## 🎮 Main Features

- Basic platformer mechanics (movement, jumping, falling)
- Simple enemy AI and obstacle challenges
- Collectibles to increase score
- UI: Main Menu, In-game HUD, Game Over screen
- Level/world system to explore and progress

## 🛠 Technologies Used

- Unity 6000.0.54f1
- C# programming
- Git & GitHub for source control

## 🚀 How to Run

1. Clone the repository:


2. Open the project in Unity (with the correct Unity version)

3. Press Play in the Editor, or build the game (PC / WebGL / Android etc.) to run


## 🎬 Inspiration

This game was inspired by the YouTube video Pirate Platformer Game.  
The video's pirate-themed gameplay, creative level design, and adventurous atmosphere motivated the development of Super Pirate World.

## 💡 Notes

- The repository does not include heavy folders like Library/, Temp/, Builds/ to keep the project lightweight. Unity will regenerate them when opening the project.
- For large assets (images, audio, etc.), use Git LFS.
